# BEMM465 Environmental Analytics

## Project Title
Applying Environmental Analytics to Support a Net-Zero UK Agrifood Sector

This repository contains the datasets and R scripts used to produce the analyses and visualisations presented in the report.

## Repository Contents

### Raw datasets (`raw_files/`)

- crops_production.csv
- emissions_crops.csv
- emissions_crops_aggregate.csv
- emissions_livestock.csv
- emissions_livestock_aggregate.csv
- emissions_prepost.csv
- emissions_prepost_aggregate.csv
- emissions_total.csv
- emissions_total_aggregate.csv

### Clean datasets (`clean_files/`)

- emissions_crops_clean.csv
- emissions_crops_aggregate_clean.csv
- emissions_livestock_clean.csv
- emissions_livestock_aggregate_clean.csv
- emissions_prepost_clean.csv
- emissions_prepost_aggregate_clean.csv
- emissions_total_clean.csv
- emissions_total_aggre_clean.csv
- gas_share_latest_year.csv
- regression_data.csv

### R scripts

- crops.r – Crop production and emissions analysis
- livestock.r – Livestock emissions analysis
- prepost.r – Pre- and post-agricultural supply chain emissions analysis
- emissions_aggregate.r – Aggregate emissions analysis
- emissions_total.r – Total agrifood emissions analysis
- gas_shares.r – Energy use and fuel mix analysis
- regression.r – Multiple linear regression and diagnostic tests

## Data source

Food and Agriculture Organization of the United Nations (FAOSTAT)
